## ⚡ Technical Spotlight

### Modern Design System (Tokens)
Instead of using hardcoded values, I implemented a robust **Design Token** system using CSS variables. This ensures consistency across the entire application and allows for instant theme switching or global branding updates.

```css
:root {
  --accent: #2563EB;
  --accent-cyan: #06B6D4;
  --bg-page: #F3F4F6;
  --shadow-card: 0 4px 24px rgba(0, 0, 0, 0.07);
  --easing: cubic-bezier(0.22, 1, 0.36, 1);
}
```

### Motion Design
I prioritize "meaningful motion"—animations that guide the user's eye rather than distracting them. I utilize a combination of custom keyframes for persistent ambient motion and Framer Motion for scroll-triggered events.

> [!TIP]
> **Key Achievement:** Implemented a lightweight `reveal` observer that triggers animations only when elements enter the viewport, ensuring high performance even on low-end devices.
